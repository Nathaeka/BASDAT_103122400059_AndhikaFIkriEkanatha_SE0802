## Soal
1. Buat Tabel Mahasiswa yang beratribut (id, nama, tempat lahir, tanggal lahir,
nomor hp, email, tinggi badan, berat badan.)
2. Lakukan insert data pada table nomor 1 sebanyak 15 rows.
3. Lakukan Fungsi (Fungsi Karakter dan String, Fungsi Tanggal dan Waktu, Fungsi Konversi, dan Fungsi Numerik)

## Deskripsi Output
1. Tabel Mahasiswa dibuat untuk menyimpan data dengan atribut id, nama, tempat_lahir, tanggal_lahir, nomor_hp, email, tinggi_badan, dan berat_badan. Setiap atribut menggunakan tipe data yang sesuai, seperti teks, angka, atau tanggal, guna menyiapkan struktur penyimpanan sebelum penginputan ke database.

2. Tahap ini melibatkan penambahan (insert) 15 baris data ke tabel Mahasiswa. Setiap baris mencakup detail lengkap seperti id, nama, kontak, serta data fisik mahasiswa. Pengisian ini bertujuan menyediakan data untuk pengolahan lebih lanjut melalui query SQL.

3. Lakukan Fungsi
    - **Fungsi karakter dan string**, Fungsi yang memungkinkan pengolahan data teks seperti nama dan email. Kegunaannya meliputi manipulasi huruf, pengambilan fragmen teks, dan penghitungan jumlah karakter untuk menghasilkan output yang sesuai kebutuhan.
    - **Fungsi tanggal dan waktu**, Fungsi yang digunakan untuk mengolah data berbasis waktu, seperti tanggal lahir. Fitur ini mencakup tampilan tanggal saat ini, perhitungan selisih, hingga penambahan bulan untuk menghasilkan output format tanggal atau numerik yang relevan.
    - **Fungsi Konversi**, Fungsi untuk mengubah tipe data, seperti teks ke angka atau tanggal, untuk menyesuaikan format data dengan kebutuhan query. Fungsi ini menghasilkan output dengan tipe atau format baru sesuai kebutuhan pengolahan data.
    - **Fungsi Numerik**, Fungsi yang digunakan untuk perhitungan data angka, seperti tinggi atau berat badan mahasiswa. Fungsi ini memungkinkan pencarian nilai rata-rata, ekstremum, atau pembulatan. Output yang dihasilkan berupa hasil kalkulasi berdasarkan data tabel tersebut.
